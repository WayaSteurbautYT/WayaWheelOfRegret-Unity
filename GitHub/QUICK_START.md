# Quick Start Guide

## Clone & Play - 3 Steps

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/WayaWheelOfRegret-Unity.git
cd WayaWheelOfRegret-Unity
```

### 2. Open in Unity
- Open Unity Hub
- Click "Open project"
- Navigate to folder
- Click "Select Folder"

### 3. Press Play!
Game starts automatically.

## What's Included

- Complete Unity project
- All 5 scenes ready
- All scripts included
- Test suite included
- No setup required

## Test Everything

1. Open any scene
2. Press Play
3. Check Console
4. All tests should pass

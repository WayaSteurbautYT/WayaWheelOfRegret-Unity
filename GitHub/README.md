# 🎮 Waya's Wheel of Regret - Unity Version

**Just clone and play!** No Unity setup required - everything is pre-configured.

## 🚀 Quick Start (3 Easy Steps)

### Step 1: Clone the Repository
```bash
git clone https://github.com/yourusername/WayaWheelOfRegret-Unity.git
cd WayaWheelOfRegret-Unity
```

### Step 2: Open in Unity
1. Open Unity Hub
2. Click **"Open project"**
3. Navigate to the cloned folder
4. Click **"Select Folder"**

### Step 3: Press Play!
That's it! The game will start automatically.

## 🆘 Troubleshooting

**"Can't open project"**
- Make sure you have Unity 2022.3 LTS or later
- Try opening Unity Hub first, then open project

**"Missing components"**
- Unity will automatically import everything
- If you see red errors, just wait for Unity to finish importing

**"Tests failing"**
- Check Console window for specific errors
- Most issues resolve by restarting Unity
